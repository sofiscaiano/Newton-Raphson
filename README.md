# NEWTON RAPHSON

El metodo de Newton o Newton-Raphson, es una de las tecnicas numericas para resolver un problema de busqueda de raices f(x)=0 mas poderosas y conocidas. Las aproximaciones se obtienen usando tangentes sucesivas y el mismo puede derivarse de los Polinomios de Taylor, lo cual subraya la importancia de una aproximacion inicial exacta, donde puede obtenerse mediante otro metodo de aproximacion como puede serlo el Metodo de Biseccion. El metodo de Newton es una herramienta muy poderosa, pero tiene un gran problema: la necesidad de conocer el valor de la derivada de f en cada aproximacion, lo cual puede evitarse realizando una peque√±a variante, que da origen al Metodo de la Secante.
